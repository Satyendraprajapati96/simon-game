# simon-game 
An interactive memory-based game built using HTML, CSS, and JavaScript, where players repeat an increasingly complex sequence of colors.

Features:- 1. Random sequence generation
           2. Increasing difficulty with each level
           3. 

How to Play: 
           1. Press any key to start the game
           2. Watch the sequence of colors carefully
           3.Repeat the same sequence by clicking the buttons
           4.Each level adds a new color to the sequence
           5.Game ends if you click the wrong sequence

               1.DOM manipulation in JavaScript
               2.Event handling (keypress & click events)
               3.Logic building and sequence tracking
               4.Improving UI with CSS animations
